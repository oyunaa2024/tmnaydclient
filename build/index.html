Апп-н тухай:

Энэхүү апп нь скада сүлжээний төхөөрөмжүүд болон бусад IP сүлжээнд ажилладаг төхөөрөмжүүдийг active эсвэл inactive байгааг PING хүсэлт илгээн хялбар байдлаар шалгадаг.

Апп ашиглах заавар:

IP хаяг гэсэн хэсэгт сүлжээний хаяг эсвэл домайн нэрийг бичиж шалгах товч дээр дарна. Мөн ангилал хэсгээс скада сүлжээний төхөөрөмжүүдийг сонгох боломжтой.
